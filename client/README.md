# Quick Chat 🗨️

A real-time MERN chat app using React, Socket.IO, Express, and MongoDB.

## 📦 Features
- Real-time messaging
- Authentication
- User presence tracking
- CRUD for users & chats

## 🛠️ Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- DB: MongoDB
- Real-time: Socket.IO

## 🚀 How to Run

```bash
cd client
npm install
npm run dev
